# Color Palette Documentation

## Premium Tech Blue Theme

This color palette is designed to be **captivating, elegant, and compelling** - making visitors want to buy your services even if they didn't initially plan to.

### Color Psychology

- **Deep Navy (#0A192F)**: Creates trust, professionalism, and sophistication
- **Electric Blue (#00D9FF)**: Represents innovation, technology, and forward-thinking
- **Gold/Amber (#FFB800)**: Conveys premium quality, success, and value
- **White/Off-white**: Provides clean, modern, spacious feeling

### Primary Colors

```scss
--color-primary: #0A192F        // Deep Navy - Main brand color
--color-primary-dark: #050A17   // Darker navy for depth
--color-primary-light: #172A45  // Lighter navy for variations
```

**Usage**: Headers, navigation, primary buttons, important sections

### Secondary Colors

```scss
--color-secondary: #00D9FF      // Electric Blue - Innovation
--color-secondary-dark: #00B4D8 // Darker blue for hover states
--color-secondary-light: #33E0FF // Lighter blue for highlights
```

**Usage**: Links, CTAs, accents, hover states, tech-forward elements

### Accent Colors

```scss
--color-accent: #FFB800         // Gold/Amber - Premium feel
--color-accent-dark: #E6A500    // Darker gold
--color-accent-light: #FFC633   // Lighter gold
```

**Usage**: Special highlights, premium features, success indicators, important CTAs

### Neutral Colors

```scss
--color-white: #FFFFFF
--color-off-white: #F8F9FA
--color-light-gray: #E5E7EB
--color-gray: #9CA3AF
--color-dark-gray: #4B5563
```

**Usage**: Backgrounds, borders, subtle text, dividers

### Text Colors

```scss
--color-text-primary: #1F2937   // Main text
--color-text-secondary: #6B7280 // Secondary text
--color-text-light: #FFFFFF     // Text on dark backgrounds
```

### Gradient Combinations

1. **Primary Gradient**: `linear-gradient(135deg, #0A192F 0%, #172A45 100%)`
   - Use for hero sections, headers, premium sections

2. **Accent Gradient**: `linear-gradient(135deg, #00D9FF 0%, #FFB800 100%)`
   - Use for CTAs, special highlights, eye-catching elements

### Usage Guidelines

#### For Maximum Impact:

1. **Hero Sections**: Use dark navy background with electric blue text/accents
2. **CTAs**: Use electric blue or gold with hover effects
3. **Premium Features**: Highlight with gold accents
4. **Tech Elements**: Use electric blue to show innovation
5. **Trust Elements**: Use navy for professional sections

#### Color Combinations That Work:

- Navy + Electric Blue = Professional + Innovative
- Navy + Gold = Premium + Trustworthy
- Electric Blue + Gold = Modern + Premium
- White + Navy = Clean + Professional
- Dark Navy + Electric Blue + Gold = Maximum Impact

### Alternative Palettes

If you want to try different palettes, we can easily switch to:

1. **Modern Purple Gradient** (Creative, innovative)
2. **Sophisticated Teal & Navy** (Growth-oriented, modern)
3. **Custom palette** (Your choice)
